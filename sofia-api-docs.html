SofIA
API Docs

Integrate SofIA into your platform. Chat, image generation, speech-to-text, text-to-speech — all in one API.

Base URL https://sofia-api-z8nr.onrender.com
01

Authentication

Authenticated endpoints require a JWT Bearer token in the request header.

Authorization header
Authorization: Bearer <your_token>
Obtain your token via POST /auth/login or POST /auth/register. Guest endpoints do not require auth.
02

Chat

Send a message and receive a streamed response via SSE. Supports text and image input.

POST /chat 🔐 auth · SSE
Body — application/json
FieldTypeRequiredDescription
messagestringrequired*Text message. Required if no image.
conversation_idstringoptionalContinue an existing conversation. Creates new if omitted.
image_base64stringoptionalBase64 image. Accepts raw or data URI format.
image_media_typestringoptionalImage MIME type. Default: image/jpeg.
SSE Response
// streaming chunks
data: {"text": "chunk..."}

// final event
data: {"done": true, "conversation_id": "abc123", "remaining_messages": 42}
For Framez: when @sof.ia is mentioned, send the post/comment content as message. Post the reply when done: true.
03

Guest Chat

Unauthenticated chat for public or anonymous users. No history persisted.

POST /chat/guest no auth
Body — application/json
FieldTypeRequiredDescription
messagestringrequiredText message to send.
Guest sessions have stricter rate limits. Conversation history is not saved.
04

Speech to Text

Transcribe audio using Whisper via Groq. Supports audio messages — useful for voice mentions on Framez.

POST /stt 🔐 auth
Body — multipart/form-data
FieldTypeRequiredDescription
audiofilerequiredAudio file. Formats: webm, mp4, mp3, wav, ogg, m4a. Max 25 MB.
languagestringoptionalLanguage code. Default: "pt".
Response
{ "text": "Transcribed content here" }
05

Text to Speech

Convert text to spoken audio using Edge TTS.

POST /tts 🔐 auth
Body — application/json
FieldTypeRequiredDescription
textstringrequiredText to convert to speech.
voicestringoptionalEdge TTS voice name. See GET /tts/voices.
06

Voice Mode

Full pipeline in a single call: STT → Chat → TTS. Send audio, receive audio + text back.

POST /voice 🔐 auth
Body — multipart/form-data
FieldTypeRequiredDescription
audiofilerequiredUser audio. Same formats as /stt. Max 25 MB.
conversation_idstringoptionalContinue an existing conversation.
languagestringoptionalAudio language code. Default: "pt".
voicestringoptionalEdge TTS voice for the response audio.
Response
{
  "text_input":      "what the user said",
  "text_response":   "SofIA's reply",
  "conversation_id": "abc123",
  "audio_base64":    "<mp3 in base64>"
}
07

Image Generation

Generate images using FLUX via OpenRouter. Requires the user's own OpenRouter API key (BYOK).

POST /generate-image 🔐 auth · SSE
Body — application/json
FieldTypeRequiredDescription
messagestringrequiredImage description / prompt.
openrouter_keystringrequiredUser's OpenRouter API key (sk-or-...).
conversation_idstringoptionalLink to an existing conversation.
SSE Response
// heartbeat
data: {"status": "generating"}

// result
data: {"done": true, "image_url": "https://...", "prompt_refined": "..."}
SofIA refines the prompt before sending to FLUX and applies a watermark to the output image.
08

History

Access and manage conversation history.

GET/history🔐 auth

Returns all conversations for the authenticated user.

GET/history/:conversation_id🔐 auth

Returns all messages in a specific conversation.

DELETE/history/:conversation_id🔐 auth

Deletes a conversation and all its messages.

09

Health Check

Verify API availability.

GET/healthno auth
